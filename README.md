### Termy48
#### A $2048$ clone for the terminal

Create games of $2048$ of any size to play in the comfort of your terminal.

#### Controls

| Keys | Moves |
|----| ---|
| w, k, :arrow_up: | Slide Up |
| s, j, :arrow_down: | Slide Down |
| a, h, :arrow_down: | Slide Left |
| d, l, :arrow_down: | Slide Right |
|q, Ctrl+c | Quit|
